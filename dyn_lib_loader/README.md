# dyn_lib_loader

A pure [V](https://vlang.io/) implementation that dynamically Load a library during runtime in scenarios where the library to load does not have a determined path and can be located in different places.

### Build and Run

```bash
$ cd dyn_lib_loader
```

```bash
$ v run .
```
