# Haversine

A pure [V](https://vlang.io/) implementation of Haversine formula.


### Build library

```sh
$ cd haversine
```

```sh
$ v . build -shared -prod -o haversinelib1
```

### Build cli

```sh
$ v -prod .
```
### Test

```bash
$ v test .
```
