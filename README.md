# V lang exercises

A pure [V](https://vlang.io/) exercises.


## Requirements

 - [V lang](https://vlang.io/) **0.5.0** and above


## Getting Started

### Clone the repo and make it yours:

```bash
$ git clone git@github.com:HomeInside/vlang-exercises.git
```


### Format

```bash
$ v fmt -w .
```


### Run example

```bash
$ v run NAME_EXAMPLE.v
```
