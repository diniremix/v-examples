# V lang exercises

A pure [V](https://vlang.io/) exercises.


## Requirements

 - [V lang](https://vlang.io/) **V 0.5.0 e2f5d6c** and above


## Getting Started

### Clone the repo and make it yours:

```bash
$ git clone git@github.com:diniremix/v-examples.git
```


### Format

```bash
$ v fmt -w .
```


### Run example

```bash
$ v run NAME_EXAMPLE.v
```
