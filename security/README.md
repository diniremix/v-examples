# módulos de criptografía y hashing

contiene ejemplos que demuestran el uso de diversas técnicas criptográficas y de hashing.

Los ejemplos están organizados en algunas categorías: 

- [x] Hashing: `hash_demo.v` que incluye ejemplos con SHA256, SHA512 y BLAKE3
- [] Derivación de Claves: `crypto_demo.v` ejemplos con PBKDF2 y Bcrypt
- [x] Aleatoriedad: `rand_demo.v` ejemplos de uso rand y UUID.
- [x] OTP: `otp_demo.v` genera claves de un solo uso (usando la librería `Meeds122.totp`)
